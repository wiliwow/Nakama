## ✅ Setup Script Created

**File:** `scripts/setup-dev.ps1` (PowerShell) and `scripts/setup-dev.sh` (Bash)

Run with optional flags:
```powershell
.\scripts\setup-dev.ps1              # Full setup
.\scripts\setup-dev.ps1 -SkipDocker    # Skip Docker (Stash service)
.\scripts\setup-dev.ps1 -SkipOllama    # Skip Ollama model check
.\scripts\setup-dev.ps1 -SkipRustBuild   # Skip Rust build (for quick env setup)
```

### What the script does:
1. Sets all environment variables from `stash.env` to `.env`
2. Creates `~/.nakama_config`, `~/.lancedb`, `~/.nakama_logs` directories
3. Installs Node.js dependencies (`pnpm install`)
4. Starts Stash Docker service (optional)
5. Verifies Ollama and pulls `deepseek-r1:1.5b` model (optional)
6. Runs cargo check with swiftide_integration feature (optional)
